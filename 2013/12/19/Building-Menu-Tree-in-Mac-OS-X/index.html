Building MenuTree in Mac OS

前言

之前见到别人用用一个目录树来显示文件夹里面的文件结构, 非常有条理, 一问才知道是Ubuntu自带的命令.
Mac OS虽然没有只带这个命令, 不过可以通过homebrew安装.

使用homebrew安装tree

brew install tree

使用方式

终端进入你需要生成目录树的文件目录, 输入命令tree即可. 效果:
