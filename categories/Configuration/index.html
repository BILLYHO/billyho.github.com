Configure GoAgentX

GoAgentX开机权限

GoagentX -> Proxy -> Setting,
勾选Fixed PAC Service Port,
填上一个端口, 如9999, 按 Restart PAC Server
然后将PAC Server 的网址复制下来
Mode 选择 Standalone

System Preference -> Network -> Advanced -> Proxies
勾选Automatic Proxy Configuration
将复制的网址填到 URL的框里面

OK -> Apply
Done!
